Module 2 - Number Properties
Overview
This module focuses on loops and functions, introducing key number theory concepts such as:

Prime Numbers
Composite Numbers
Abundant Numbers
Narcissistic Numbers
The goal is to write Python code that identifies different number properties efficiently.

Learning Objectives
Understand and implement loops in Python.
Define and use functions to simplify code and improve reusability.
Apply number theory concepts in programming.
Write clear comments and docstrings for better code readability.
